# IBM Data Science Project: Survey Data Insights

## Project Overview
This project involved analyzing survey data to identify key trends. I used **Python (Pandas)** for data cleaning and **IBM Cognos** for professional dashboarding.

## 🛠 Tools & Libraries
- **Python:** Pandas, NumPy
- **Visualization:** IBM Cognos Analytics
- **Project Goal:** To analyze user demographics and tech preferences.
